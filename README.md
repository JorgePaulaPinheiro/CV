# Paula Pinheiro Jorge - Portfolio

A modern React portfolio website showcasing my transition from IT to Economics & Data Analytics.

## 🚀 Features

- **Modern Design**: Dark theme with glassmorphism effects
- **Responsive**: Works on desktop, tablet, and mobile
- **Fast**: Built with Vite for optimal performance
- **Auto-deploy**: GitHub Actions automatically deploys to GitHub Pages on every push

## 🛠 Tech Stack

- React 19
- Tailwind CSS v4
- Vite 8
- Lucide Icons
- GitHub Actions for CI/CD

## 📦 Getting Started

### Prerequisites
- Node.js 20+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/jorgepaulapinheiro/CV.git
cd CV

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🚀 Deployment

The site is automatically deployed to GitHub Pages when you push to the `main` branch.

To enable this:
1. Go to your repository settings on GitHub
2. Navigate to **Pages** section
3. Set **Source** to "GitHub Actions"
4. The workflow will handle the rest

## 📝 Content Updates

To update your portfolio:
1. Edit the relevant component in `src/components/`
2. Commit and push to `main`
3. GitHub Actions will automatically rebuild and deploy
